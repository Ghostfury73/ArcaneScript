ArcaneScript Syntax

Print: `print value;`

Println: `println value;`

Assignment: `variable = value;`

Array Initialization:

array name[size] = {value1, value2, ...};

array name[size];

array name[] = {value1, value2, ...};

For Loop:

Syntax: `for (initialization condition increment) { statements }`

Example: `for (i = 0; i < 10; i = i + 1;) { print i; }`

While Loop: `while (condition) { statements }`

If Statement:

`if (condition) { statements }`

`if (condition) { statements } else { statements }`

Method Declaration: `invoke methodName(parameters) { statements }`

Log: `log value;` (similar to `return`)
